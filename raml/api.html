/node/{nodeid}/process/{proccessid} get delete

get /node/{nodeid}/process/{proccessid}

Get process details

  • Request
  • Response

URI Parameters

  • nodeid: required (string)
  • proccessid: required (string)

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • cmd: required (object)

    Job that is be executed in the Node/Container. More information can be found here https://github.com/g8os/node/blob/master/docs/commands.md

    • id: required (string)

      A unique id for the job with which later on results can be retrieved. If not supplied an id will be generated.

    • queue: required (string)

      Name of the queue in which this job was be enqued.

    • tags: required (string)

      Metadata that is be submitted together with the job and is returned with the result.

    • statsInterval: required (integer - default: 0)

      Stats gathering interval.

    • maxTime: required (integer - default: 0)

      Max run time of the job, if exceeded job will be killed

    • maxRestart: required (integer - default: 0)

      Max number of retries to start the job if failed before giving up.

    • recurringPeriod: required (integer - default: 0)

      If provided job is executed every X seconds.

    • logLevels: required (array of integer - default: 0)

      Log levels to store locally and not discard.

  • cpu: required (number)
  • debug: required (string)
  • rss: required (number)
  • swap: required (number)
  • vms: required (number)
