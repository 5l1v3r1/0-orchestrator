Body

Media type: application/json

Type: object

Properties
  • id: required (string)

    The unique identifier with which the command was executed

  • name: required (one of core.ping, core.system, core.kill, core.killall, core.state, core.reboot, info.cpu, info.disk, info.mem, info.nic, info.os, corex.create, corex.list, corex.dispatch, corex.terminate, bridge.create, bridge.list, bridge.delete, disk.list, disk.mktable, disk.mkpart, disk.rmpart, disk.mount, disk.umount, btrfs.create, btrfs.list, btrfs.subvol_create, btrfs.subvol_list, btrfs.subvol_delete)

    The kind of command to which this result belongs to.

  • level: required (string)

    20 means that the data is json serialized object

  • starttime: required (integer)

    Epoch timestamp when the command was started

  • stdout: required (string)

    Stdout of process/command

  • stderr: required (string)

    Stderr of process/command

  • state: required (one of SCHEDULED, RUNNING, SUCCESS, KILLED, KILLED_TIMEOUT, FAILED)

    Status of the command

  • data: required (string)

    The result of executing the command
